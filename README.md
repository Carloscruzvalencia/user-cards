## Requisitos

### Instalacion
sass

via npm
```shell
npm install -g sass
```
via Chocolatey
```shell
choco install sass
```

### En desarrollo
```shell
sass --watch assets/data-files/sass/styles.scss assets/data-files/css/custom.css
```
